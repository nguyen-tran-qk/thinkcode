Typography

Great typography is the core of any user interface design.

Material Design Typographic Guidelines

We have followed the material design typographic guidelines to work across the wider set of supported platforms.

Too many type sizes and styles at once can wreck any layout.

A typographic scale has a limited set of type sizes that work well together along with the layout grid.

The basic set of styles are based on a typographic scale of 12, 14, 16, 20 and 34.
